# Abacus for Tilda

Интерактивный абакус (соробан) для встраивания в Tilda. Один скрипт обслуживает
любое количество абакусов на странице — без конфликтов `id` и глобальных переменных.

Демо-страница оформлена в эстетике [shadcn/ui](https://ui.shadcn.com) и выдаёт
готовый код для вставки в HTML-блоки Tilda.

## Возможности

- Клик по косточкам, верхняя косточка = 5, нижние = по 1.
- Кнопка **Подсказка** показывает значение на каждой спице.
- Кнопки **+/−** меняют количество спиц (1–12).
- Кнопка **Сброс** обнуляет состояние.
- Любое количество абакусов на одной странице, полностью изолированы друг от друга.

## Структура

```
.
├── index.html        — демо-страница / генератор кода
├── css/
│   ├── page.css      — стили демо-страницы (shadcn). В Tilda не нужны.
│   └── abacus.css    — стили самого абакуса. Нужны в Tilda.
└── js/
    ├── page.js       — логика генератора (сборка кода, копирование, тема)
    └── abacus.js     — движок абакуса. Нужен в Tilda.
```

## Локальный запуск

Страница подтягивает `abacus.css` и `abacus.js` через `fetch`, поэтому открывать
её нужно через сервер, а не как `file://`:

```bash
npx serve
# или
python3 -m http.server
```

## GitHub Pages

В настройках репозитория включите **Settings → Pages → Source: main /(root)**.
Демо будет доступно по адресу `https://<username>.github.io/<repo>/`.

## Как встроить в Tilda

1. Добавьте блок **T123 → HTML** (подходит и Zero-блок).
2. В **первый** такой блок вставьте код из **Шага 1** на демо-странице
   (стили `<style>` + скрипт `<script>`). Он нужен один раз на странице.
3. Для **каждого** абакуса добавляйте новый HTML-блок с кодом из **Шага 2**.
4. Сохраните и **опубликуйте** страницу.

> **Важно.** Минификатор Tilda ломает компактный JS (например `for(...)` без
> фигурных скобок). Код намеренно развёрнут в полные блоки, чтобы это пережить —
> не сокращайте его вручную.
